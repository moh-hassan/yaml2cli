# yaml2cli
yaml2cli is a dotnet global Generate System.Commandline commands, options and arguments from Yaml string.


To try `yaml2cli`, download the package from github  [Yaml2Cli.1.0.0-beta.1.nupkg](https://github.com/moh-hassan/yaml2cli/releases/download/1.0.0-beta.1/Yaml2Cli.1.0.0-beta.1.nupkg)

Install the package as global tool like:

```
dotnet tool install -g yaml2cli  --version 1.0.0-beta.1  --add-source ./path/to/download/folder
```
